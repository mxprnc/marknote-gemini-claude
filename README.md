# Gemini CLI
리포지터리 이름이 Claude 가 들어가지만 Claude 본격적으로 다루는 리포지터리는 아닙니다. Claude Code 의 주요 기능을 Gemini CLI 에서 지원하는지 등을 조사해서 개념들을 정리하고 예제를 정리해두는 곳 입니다. 취업을 위해 공개리포지터리로 해두었지만, 언젠가는 비공개로 해둘 예정입니다.

- docs : 공식문서 요약한 디렉터리입니다.
- 그 외 : 개인적인 연습 내용들 또는 학습내용입니다.


최근에는 Gemini CLI 를 일정 Step 과 상태를 이용해서 루프로 돌도록 하면서 테스트와 검증을 반복하는 과정에 관심을 가지고 있고 그와 관련된 템플릿/빌더 를 만드는 작업을 비공개 리포지터리로 진행중입니다. 공개할만한 수준이 된다면 이 곳에도 에제를 설명하는 문서를 남겨두도록 하겠습니다.
