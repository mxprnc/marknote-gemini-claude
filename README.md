# Gemini CLI, Claude Code 

Claude Code 의 주요 개념들을 정리하고, Gemini CLI에서 비슷한 개념은 무엇이 있고 어떻게 더 효율적으로 쓸수 있는지를 정리합니다.<br/>

Claude 에 비해 Gemini 요금이 요즘 너무 싸고 flash 모델의 효율이 가성비가 너무 높고 컨텍스트 소모도 적고 허용되는 컨텍스트 용량도 큰것 같아서... 시작하게 됐습니다... 그런데요... 문서 중간에 독백체가 있어서 놀라실수도 있는데요. 혼자서 정리하다가 점점 사이즈가 커져서 이렇게 되긴 했는데 아직 혼자서 정리하던 때의 문서는 손을 못대고 있습니다. 이 점 양해해주셨으면 합니다.<br/>
<br/>

생각보다 Gemini CLI 의 기능이 유용한 것들이 많고 Claude Code 의 명령어와 호환되는 것도 많아서 가볍게 볼만한 주제는 아니라고 생각했고, 어느 정도는 진지하게 업무에서 일부분을 적용해서 업무 속도를 늘릴수 있는 방법을 계속 찾아가야겠다고 생각해서 계속해서 작업을 해나가려고 합니다.<br/>
<br/>

Google 은 하드웨어도 싸게 공수할 수 있는 회사라 앞으로 토큰 허용치도 더 크게 늘어날 것 같고 요금제도 더 저렴해질수도 있지 않나 하는 생각도 들어서 남들 다 Claude Code 공부하고 스터디할 때 저는 Claude Code 스터디 내용 요약본을 혼자서 Gemini CLI 버전으로 학습해보고 정리해보고 있습니다.<br/>
